## The Queue Dilemma Project Summary
This project explores how people should choose a checkout line in a supermarket. Using a simulation, five different strategies are tested — from joining the shortest line to estimating which queue will move fastest based on item counts and cashier speed. Customers arrive randomly, carry different‑sized baskets, and each cashier has a mix of fixed overhead time and item‑scanning time. The strategies are compared by looking at average waiting time, long waits, variability, and how well each method handles crowd behavior. The goal is to understand which queue‑picking rule actually helps customers get through the checkout faster and more reliably.
